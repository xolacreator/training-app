# Hybrid Training App

A progressive web app (PWA) for tracking a 12-week hybrid training programme across Phase 1 and Phase 2.

## Features
- **Today** — live dashboard with countdown, week strip, and today's session
- **Log** — fast session logging with pace, HR, distance, duration, feel rating
- **Plan** — full 12-week programme browser with session breakdowns (warm-up → main set → cool-down)
- **Stats** — volume, pace trend, feel score, and completion charts
- **More** — Garmin CSV import, light/dark mode, data management

## Session library
Full warm-up → main set → cool-down breakdowns for all 84 days across Weeks 1–12, including:
- Option A / B / C switching per day
- Coaching cues and intensity targets
- Injury flags (left knee, right wrist, low back)
- Phase 1 (Weeks 1–6) and Phase 2 (Weeks 7–12)

## GitHub Pages setup
1. Upload `index.html` to your repository
2. Settings → Pages → Deploy from main branch / root
3. Visit your URL in Safari on iPhone → Share → Add to Home Screen

## Garmin import
Export CSV from Garmin Connect → Activities → Export CSV, then upload via the Import tab.
